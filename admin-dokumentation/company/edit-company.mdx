## Wie bearbeitet man ein Unternehmen?  

### 1. **Navigieren Sie zur Seitenleiste und klicken Sie auf „Unternehmen“**  

![Unternehmen Anklicken](/images/sidebar_Unternehmen_anklicken.png)  

Navigieren Sie zur Unternehmensübersicht, um eine Liste aller angelegten Unternehmen anzuzeigen.

### 2. **Wählen Sie das gewünschte Unternehmen aus**  

Suchen Sie das Unternehmen, das Sie bearbeiten möchten, in der Übersicht. Klicken Sie entweder auf das **Bearbeiten-Symbol** (Stift-Icon).  

![Bearbeiten Anklicken](/images/bearbeiten_anklicken.png) 

### 3. **Passen Sie die Informationen an**  

In der Bearbeitungsansicht können Sie die folgenden Felder ändern:  

#### **Pflichtfelder:**  
- **Name des Unternehmens**  
  - Ändern Sie den offiziellen Namen des Unternehmens, falls notwendig.  
- **Umsatzsteuer-ID**  
  - Tragen Sie eine aktualisierte Umsatzsteuer-Identifikationsnummer ein, falls vorhanden.

#### **Optionale Informationen:**  
- **Straße**  
  - Passen Sie die Straße ihres Unternehmens an.  
- **Postleitzahl**  
  - Geben Sie Ihre Postleitzahlan.  
- **Stadt**  
  - Verändern Sie die Stadt.  

### 4. **Speichern Sie die Änderungen**  

Klicken Sie auf die Schaltfläche **„Speichern“**, um die Änderungen zu übernehmen.  

![Änderungen Speichern](/images/aenderungen_speichern.png)  

### 5. **Das Unternehmen wird mit aktualisierten Informationen in der Übersicht angezeigt**  

Nach dem Speichern erscheinen die geänderten Informationen direkt in der Unternehmensübersicht. Sie können das Unternehmen bei Bedarf weiter bearbeiten.
