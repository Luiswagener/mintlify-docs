## Wie erstellt man einen Benutzer?  

### 1. **Navigieren Sie zur Seitenleiste und klicken Sie auf „Benutzer“**  

![Benutzer Anklicken](/images/sidebar_benutzer_anklicken.png)  

### 2. **Klicken Sie anschließend auf „Neuen Benutzer“**  

![Neuen Benutzer Anklicken](/images/neuen_benutzer_klicken.png)  

### 3. **Füllen Sie die Pflichtfelder aus, die mit einem „*“ gekennzeichnet sind:**  
- **Name**  
  - Geben Sie den vollständigen Namen des Benutzers ein.  
- **E-Mail-Adresse**  
  - Geben Sie die E-Mail-Adresse des Benutzers ein, die für die Anmeldung und Benachrichtigungen verwendet wird.  
- **Rolle**  
  - Klicken Sie auf das Kästchen "Admin" damit dieser Nutzer als ein Admin eingestuft wird

### 4. **Fügen Sie optionale Informationen hinzu:**  
- **Telefonnummer**  
  - Geben Sie die Telefonnummer des Benutzers ein.  
- **Abteilung**  
  - Falls relevant, ordnen Sie den Benutzer einer Abteilung zu.  
- **Adresse**  
  - Tragen Sie die Adresse des Benutzers ein, falls erforderlich.  
- **Notizen**  
  - Fügen Sie zusätzliche Informationen oder Kommentare zum Benutzer hinzu.  

### 5. **Klicken Sie auf „Benutzer Erstellen“**  

![Benutzer Erstellen Anklicken](/images/benutzer_erstellen_anklicken.png)  

### 6. **Der Benutzer erscheint nun in der Benutzerübersicht und kann weiter bearbeitet oder verwaltet werden.**
