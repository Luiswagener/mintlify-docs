## Wie bearbeitet man einen Benutzer?

### 1. **Navigieren Sie zur Seitenleiste und klicken Sie auf „Benutzer“**

![Benutzer Anklicken](/images/sidebar_benutzer_anklicken.png)

### 2. **Suchen Sie den Benutzer, den Sie bearbeiten möchten, und klicken Sie auf „Bearbeiten“**

![Benutzer Bearbeiten Anklicken](/images/benutzer_bearbeiten_klicken.png)

### 3. **Bearbeiten Sie die gewünschten Informationen:**

#### **Pflichtfelder:**
- **Name**  
  Aktualisieren Sie den vollständigen Namen des Benutzers, falls erforderlich.
- **E-Mail-Adresse**  
  Passen Sie die E-Mail-Adresse an, wenn diese geändert werden muss.
- **Rolle**  
  Überprüfen und ändern Sie die Rolle des Benutzers. Aktivieren Sie z. B. das Kästchen "Admin", falls der Benutzer als Administrator eingestuft werden soll.

#### **Optionale Felder:**
- **Telefonnummer**  
  Bearbeiten oder fügen Sie die Telefonnummer hinzu.
- **Abteilung**  
  Aktualisieren Sie die zugeordnete Abteilung des Benutzers, falls nötig.
- **Adresse**  
  Ändern oder ergänzen Sie die Adresse des Benutzers.
- **Notizen**  
  Überarbeiten Sie zusätzliche Informationen oder Kommentare.

### 4. **Speichern Sie die Änderungen, indem Sie auf „Speichern“ klicken**

![Speichern Anklicken](/images/speichern_klicken.png)

### 5. **Die Änderungen werden übernommen, und der Benutzer erscheint mit den aktualisierten Informationen in der Benutzerübersicht.**
