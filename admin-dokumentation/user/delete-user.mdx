## Wie löscht man einen Benutzer?

### 1. **Navigieren Sie zur Seitenleiste und klicken Sie auf „Benutzer“**

![Benutzer Anklicken](/images/sidebar_benutzer_anklicken.png)

### 2. **Suchen Sie den Benutzer, den Sie löschen möchten, und klicken Sie auf „Löschen“**

![Benutzer Löschen Anklicken](/images/benutzer_loeschen_klicken.png)

### 3. **Bestätigen Sie die Löschung des Benutzers**

- Es erscheint ein Pop-up oder eine Bestätigungsnachricht, in der Sie gefragt werden, ob Sie den Benutzer wirklich löschen möchten.  
- **Hinweis:** Die Löschung ist dauerhaft und kann nicht rückgängig gemacht werden.  
- Klicken Sie auf **„Ja, löschen“**, um die Aktion zu bestätigen.

### 4. **Der Benutzer wird aus der Benutzerübersicht entfernt**

Nach der Bestätigung wird der Benutzer dauerhaft gelöscht und erscheint nicht mehr in der Benutzerübersicht.

### 5. **Prüfen Sie die Benutzerübersicht**

Überprüfen Sie die Benutzerliste, um sicherzustellen, dass der Benutzer erfolgreich entfernt wurde.
